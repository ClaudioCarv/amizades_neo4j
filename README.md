
# Bancos NoSQL orientado a grafos - Neo4j

Neste projeto, foi realizado a utilização do Neo4j, um banco de dados NoSQL orientado a grafos onde foi utilizado o conceito de amizades para as relações dos nós.



## Instalação

Instale e importe as bibliotecas necessárias

```bash
  pip install neo4j
  import uuid
  import json
```
    
## Demonstração

primeiro crie uma instancia no AuraDB, após isso abra o arquivo "tokens" e coloque o usuario e a senha disponibilizado pela AuraDB.
 Após isso é necessário apenas executar o arquivo "main.py".
  No Menu você terá várias opções para executar a primeira seria a da criação de uma pessoa, após adicionar nome, idade e telefone, será criado um nó na sua instancia no AuraDB, após isso você pode criar outra pessoa e criar uma amizade entre elas, selecionando a opção de criar uma amizade e passando os IDs das duas pessoas.
